How to Log Food by Text

Type what you ate. KiloKaki parses it and calculates macros instantly.

Text logging is the most flexible input. If you can type it, KiloKaki can parse it. Single ingredients, full meals, multiple items at once — it's all fair game.

Step by Step

1

Open KiloKaki in Telegram

Start a message in the KiloKaki chat. You don't need a slash command or special format — just type naturally.

2

Type what you ate

Write it the way you'd describe it to someone. Include quantities where you know them.

Text message "2 eggs 1 slice whole wheat toast with butter"
Text message "300g grilled salmon 1 cup brown rice cucumber"
Text message "1 bowl wanton noodle soup no prawn"
3

Send

KiloKaki parses your text and replies with a macro breakdown. If you mentioned multiple items, they'll be itemised.

💡 Text works great for meals you eat regularly. If you have a go-to lunch, you can type it in seconds — or save it as a template for one-tap logging. See: How to Use Meal Templates

Text Format Tips

  • Use commas or line breaks between separate items — "rice, chicken, veg" is clearer than "rice chicken veg"
  • Include units when you know them — "200g chicken breast", "1 cup rice", "30g cheese"
  • Specific dish names work — "beef rendang with rice" is fine, doesn't need to be broken into ingredients
