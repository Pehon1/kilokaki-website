How to Use Meal Templates

Save your go-to meals. Log your usual lunch in seconds — one tap.

If you eat the same things regularly, meal templates save you from typing them out every single time. Set it up once, use it forever.

Creating a Template

1

Log the meal normally first

Use voice, text, or photo to log the meal exactly as you would normally. This gives KiloKaki the macro data it needs.

2

Save it as a template

Reply to the logged entry and tell KiloKaki to save it:

Reply to a log entry "save as template" or "save as lunch template"

Give the template a short, memorable name — like "usual lunch" or "work breakfast."

3

KiloKaki confirms

The bot confirms the template is saved and tells you how to recall it.

Using a Template

When you want to log your saved meal:

Text message "log my lunch template"

Or just the template name:

Text message "usual lunch"

KiloKaki pulls the saved meal and logs it instantly with the same macro values.

💡 Templates are perfect for: work-day lunches, gym-day breakfasts, post-workout shakes, weekly meal prep. If it's the same most days, make it a template.

Managing Templates

  • List your templates by asking: "show my templates"
  • Delete one by: "delete lunch template"
  • Update by logging the new version and re-saving with the same name
