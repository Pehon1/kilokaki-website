How to Create a Custom Food

When KiloKaki doesn't recognise a food, add it yourself with your own macro values.

KiloKaki's food database covers a wide range of dishes and ingredients, but not everything. If you log something and KiloKaki can't identify it — or gives you a poor estimate — you can add it manually as a custom food.

When to Create a Custom Food

  • A dish that isn't in any standard database
  • Home-cooked recipes you make regularly
  • Local foods, specialty items, or branded products not in USDA data
  • Ingredients you combine in specific ratios

Step by Step

1

Log the food as usual

Try logging the food via voice, photo, or text. If KiloKaki doesn't recognise it, it'll tell you.

2

Send the food details manually

If KiloKaki can't match the food, type the food name and macros directly:

Text message "homemade chicken curry 350 cal 30g protein 40g carbs 8g fat"

Include the name and any macro values you know. KiloKaki will log it with the values you provide.

3

Correct values if needed

If the estimate isn't right, tap ✏️ Edit on the entry and adjust the macros using the Change Macros option.

4

Save it for next time (optional)

If you eat this regularly, tap 💾 Save as template so you can log it in one tap next time. Learn more →

💡 Find macro values on the nutrition label, restaurant website, or a recipe calculator. Even an estimate is better than nothing — your body learns the pattern over time.
